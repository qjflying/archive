# 知识管理工具

> 通过文件夹分类存放管理知识, 配置目录关联文件, 支持markdown、图片、pdf、xmind等多种文件格式。

<a target="_blank" href="https://qjflying.github.io/archive/demo/index.html">点击查看演示</a>

## 目录配置
* 使用文件夹
    ```
    * **基础格式**
    ```
* 文件列表
    ```
    * 报考指南.pdf
    * 测试.md
    * config.js
    * hello.html
    * js.gif
    * 清单革命.xmind
    * [外部链接](//www.baidu.com/)
    ```    
* include
    ```
    * @include:include文件夹a/list.md   
    ```
* 关联参数
    ```
    * 测试文件d|@title:改名+改类型|@type:iframe
    ```
